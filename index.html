Synthetic Tensor Generation

Controlled sparse tensor generators for experiments.

TensorSuite writes .tns coordinate-format tensors whose non-zero values are sampled from a standard normal distribution. The limited generator estimates memory use before writing data.

Open SynTensorGen code
Single tensor python3 SynTensorGen/random_tensor_generator_limited.py output.tns D 2.0 50%1024 2%4096
Preset suite bash SynTensorGen/generate_all_tensors_limited.sh

Output Format

.tns format notes and header documentation.

This section is reserved for detailed format documentation. The current synthetic generator writes a compact header before the coordinate entries; future notes can define required fields, optional metadata, compatibility with external datasets, and validation rules.

Header

Line 1: tensor order. Line 2: mode dimensions.

Entries

Each data row stores one-based coordinates followed by a value.
